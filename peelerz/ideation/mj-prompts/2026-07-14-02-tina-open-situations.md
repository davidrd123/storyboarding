**Date:** 2026-07-14  
**Purpose:** test the actual openness of Ben's brief. Every prompt preserves
only Tina, the back of the car, the candy, and a communicable creative
situation. Each prompt changes a different category rather than varying one
established mechanism.

Use the original Tina-in-car image as an identity reference when useful, but
do not use the miniature-neighborhood keeper as a reference for this round.
