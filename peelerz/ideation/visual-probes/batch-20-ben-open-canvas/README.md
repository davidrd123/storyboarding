# Batch 20 — Ben open canvas

Six two-variant NB2 Lite probes selected from the July 15 open-canvas round.

Unlike Batches 18–19, this batch does not treat Tina's parachute, Raul's mecha,
or their existing transformation mechanics as destinations. Only the talent
entrances and general Peelerz register remain fixed.

The six probes test whole-world, time, authorship, scale-exchange, mythic-world,
and cosmological transformations.
