# Focused visual-probe mode

Use this mode after a creative room has both useful trunks and diagnosed gaps,
but current image work is staying too close to reference images.

## What changes

The engine's external entropy, binding obedience, phase wall, lineage, and
archive rules remain intact. Four additional fields become mandatory:

1. **GAP** — the concrete problem this exploration addresses.
2. **INHERIT** — what references lock and may not be casually mutated.
3. **RING** — how far the seed is allowed to travel from the trunk.
4. **VISUAL PROOF** — the one image or short sequence that could demonstrate
   the seed. This is description, not feasibility judgment.

## Distance rings

- **Adjacent:** preserve characters, world, action, and emotional fantasy;
  mutate only the physical mechanism or camera.
- **Lateral:** preserve characters, product behavior, and emotional fantasy;
  allow the world, medium, scale, or social reaction to change.
- **Feral:** preserve only the contractual anchors, product bridge, register,
  and emotional fantasy. A stranger must still be able to caption the visual
  proof correctly.

Per focus, generate two seeds in each ring before comparison. Never demote a
feral draw into an adjacent execution because it feels risky.

## Focus packet

Write this before drawing entropy:

```
CONCERN:
GAP:
TRUNK:
INHERIT:
MAY MUTATE:
NON-NEGOTIABLES: product present; youth-coded; benign/appetizing
OUTPUT: one still | two-frame sequence | three-frame sequence
```

## Seed format during divergence

```
RING:
DRAW:
RULE: one impossible rule
VISUAL PROOF: subject / action / spatial relation / impossibility /
              emotional read / composition
BRIDGE: product operation + exteriorized feeling
LINEAGE: operator / stimulus / parent
```

Do not rank, polish, or discuss model feasibility until every scheduled seed
exists.

## Focused pilot quota

For a first experiment:

- 3 focuses × 6 seeds (2 adjacent, 2 lateral, 2 feral)
- 18 archived seeds
- 8 Gate-1 survivors
- 6 NB2 Lite visual probes
- 3 presentation candidates at most

This smaller quota is valid only because authored trunks and diagnosed gaps
already exist. A blank-page session still uses the main engine quota.

## Gate additions

At Gate 1, ask whether the weirdness pays at least one kind of rent:

- **mechanical rent:** clarifies or solves the diagnosed transformation gap;
- **image rent:** creates a compelling, immediately stageable picture;
- **story rent:** reveals character or relationship more sharply;
- **transition rent:** creates a useful handoff or shared visual rhyme.

One kind is enough to remain available. Gate 2 still applies the full brief-
fitness rubric.

## Reference inheritance

Every render adapter labels each image as one of:

- identity authority;
- world-plate authority;
- material authority;
- mechanism authority;
- composition inspiration only.

If the seed intentionally breaks an inherited property, it must say which
property and why. Unnamed reference drift is not creative divergence.

## Gate 2.5 — render compiler

A concept seed is not an image prompt. Metaphors are useful during divergence,
but image models often literalize the metaphor noun and discard the mechanism.
Before Gate 3, compile every selected seed into a causal specification:

```
INVARIANTS: identity, world, and material properties that must survive
SUBSTRATE: the one thing physically changing
INITIAL STATE: its readable state before the change
APPLIED FORCE: who or what acts on it, at which exact contact points
TOPOLOGY: counts, connections, gaps, hinges, and continuous edges
STATE CHANGE: visible before/after difference across the active boundary
RESIDUE: what remains attached, discarded, or under tension
SHOT PROOF: the minimum composition that exposes the causal chain
LITERALIZATION HAZARDS: metaphor and style nouns likely to become props
```

Then write the render prompt from that specification with these rules:

1. Remove every metaphor named in `LITERALIZATION HAZARDS`. Describe only its
   mechanical consequence. A "lotus opening" becomes four continuous flexible
   sheets rotating around named attached edges; no flower language survives.
2. Remove source-domain agents that have no physical job. A lighthouse may
   inspire a steady directional signal, but no lighthouse, beam tableau, or
   nautical styling enters the prompt.
3. Use one impossible clause. Everything else should obey explicit contact,
   tension, continuity, scale, and gravity.
4. Prefer countable topology over evocative adjectives: four sheets, one
   uninterrupted connector, an eight-inch gap, first and last hands gripping
   named loops.
5. Add an attractor veto naming the most likely picturesque substitution.

### Attractor audit

After compilation, ask: if the metaphor words are gone, is the image still
surprising? If not, the seed was verbally strange but visually empty. Return it
to the nursery rather than asking the renderer to invent its substance.

Gate 3 compares the rendered mechanism to the compiled specification, not to
the beauty or conceptual charm of the original seed.

## Honour the render error

Every generated artifact receives one sentence: “Assume this was intended;
what useful idea is it the correct image of?” The answer may seed the next
focused batch, but it cannot retroactively count as success for the prompt.
