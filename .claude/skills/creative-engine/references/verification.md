# Verification — the staged gates

Strange candidates only need to earn **continued availability** during
divergence — not prove they're already the best answer. Verification has
levels so it never becomes conventionality policing. Only Gates 2–3 decide
what gets developed for presentation.

**Comparative judging rule (all gates):** LLM judges inherit the typicality
bias that caused the collapse. Judge in comparative batches, never absolute
isolated scores; list merits before flaws (Synectics itemized response).

## Gate 0 — admissibility (in-session, cheap → nursery)

- Relates to an identifiable concern?
- Can the transformation RULE be stated in one sentence?
- At least a plausible product connection (may be flagged `unbridged` and
  admitted anyway — a failed bridge is a flag, not a kill)?
- Meaningfully different from its parent and from everything round-banned?
- **Cliché check:** tag any seed matching `decks/cliche-blacklist.md`; a
  tagged seed must pass one repair operator (INVERT / ADD COST / RE-CHANNEL)
  before it can advance. Clichés are raw material, never final answers.

## Gate 1 — developmental potential (end of session → portfolio)

- Does elaborating the rule produce more consequences? (run REPERCUSSIONS)
- Can it generate a FRAME and a transition out?
- Does it occupy a sparse niche in the archive? (check `session.py metrics`)
- Does it carry an emotionally meaningful fantasy (escape / power /
  connection / belonging / recognition)?
- REVERSAL stress-pass: imagine it failing in production; what kills it?

## Gate 2 — brief fitness (separate context, comparative)

- **Poster test (three tiers, not pass/fail):** single-still (a stranger
  captions one mid-event frame correctly — premium strength) / sequence
  (2–3 stills communicate it: before/after pair or triptych — fully
  deliverable, must be declared as such) / motion-or-sound-only (cannot be
  boarded — demote to transition/sound note, never present as a frame idea).
  Single-still is a bonus, not a gate.
- **Contract-conditional ownability** (four imaginal contracts — every seed
  declares one; judge by ITS test, never causal-magic for all):
  - *causal* — product action physically causes the event → is the product
    visibly responsible? Snack-swap test applies.
  - *metaphoric* — a product quality becomes a literal image → is the
    translated quality specifically this product's (choice, two textures,
    inside/outside)?
  - *affective* — the film behaves the way consuming feels → what EXACT
    feeling disciplines every choice? (Gorilla's filter was strict, not
    open.) Vague "joy" fails.
  - *world-building* — the product opens a coherent alternate system →
    could this world only have been generated by this product? (Happiness
    Factory test.)
- **One rule** (contract-conditional): causal/metaphoric → one governing
  law, escalated with logic; affective → one governing feeling; world-
  building → one coherent system.
- **Two dials:** |event-magnitude − reaction-magnitude| ≥ 5.
- **Cost present:** someone quietly pays; everyone underreacts.
- **Register:** right-coded for the audience (for Peelerz: youth-coded, not
  infantile); appetite appeal intact.
- **Escalation slot:** correct scale for its position (early beats must not
  eclipse the finale).
- **One-verb test** (dossier/Bravia): the transformation names itself with
  one dominant verb — peels, unspools, assembles, reveals, snaps. Pitchable
  as one noun + one verb ("car skin unspools").
- **Visible trigger (contract-conditional):** causal → SEE the product
  action cause the event; metaphoric → legible product-quality↔image
  correspondence; affective → the precise feeling, reproduced; world-
  building → the system organized around the product. (Unconditional
  visible-trigger would reject Cadbury Gorilla — the dossier's own
  checklist makes this mistake.)
- **Invariant:** across changing worlds, what stays stable enough to orient
  the viewer (object, gesture, camera, material)?
- **Transition value:** does the frame also create the exit into the next
  scene? (These are moving sequences, not standalone posters.)
- **Platform test:** write 3 more one-line executions from the same rule.
  Can't? It's an execution, not an idea — mine it for its platform and
  regenerate. (3+ distinct executions is also the EVIDENCE required to mark
  a mechanism `protected` in the ban-list.)

## Gate 3 — render (empirical, per-model)

The final judge is a rendered frame, not an argument.

- **FRAME is model-neutral but concretely depictable** — one stageable
  tableau with mandatory fields: subject / action / spatial relation /
  governing impossibility / emotional read / composition. Never conceptual
  prose. The field list IS the legibility discipline.
- **RENDER ADAPTERS** translate FRAME into model-specific prompts only at
  this gate (MidJourney, NB2, whatever's new). Adapters keep current prompt
  conventions from shaping ideation upstream.
- **Verdicts are tagged per-model, never absolute.** An idea MJ mangles may
  be trivial for NB2 with a reference frame (refs are load-bearing — see
  repo NB2 findings: SHOW facture, don't describe it).
- **Evaluate the visible verb, not the prompt's verb** (Codex batch-02): the
  result note records what the image actually shows a stranger — "opens,"
  "disassembles," "drapes" — even when the prompt said "peels." The render
  verdict is about the screen, not the intention.
- **Two independent axes, never one score:** premise strength and product
  causality are separate observations (a frame can be the funniest read in
  the batch with weak causality, or mechanically perfect and inert).
- **Probe cheap, finalize expensive:** reaction objects in the fast model
  (Flash/1K), deck-final frames in the quality model (Pro/2K). Per-model
  verdicts get two columns, not one.
- **Honour thy error:** every artifact/wrong render gets one mandatory pass
  as a hidden intention — "assume this was intended; what is it the correct
  version of?" — a seed, not a reject. Only then re-roll.
- **Not a hospice:** repeated cross-model failure kills the treatment slot.
  The idea stays archived with its render verdicts; the slot goes to the
  next portfolio candidate.
- Harness: the `generation-review-loop` skill.

## Treatment-shape output (what leaves the engine)

For every presented idea:

```
RULE:   one sentence — the single impossible mechanism
FRAME:  the selling still (the 6 mandatory fields), + what a stranger
        would caption it
LADDER: 3 escalations of the same rule, each conceptually new, each
        citing its license
DELTA:  the one change from documentary reality (= the budget)
COST:   who quietly pays
LINEAGE: operator / stimulus / parent (from session.py)
```
